# Project-Pemrogaman-Dasar
YohanKos adalah aplikasi berbasis Python yang memungkinkan pengguna mencari dan memesan kos dengan filter harga, fasilitas, dan lokasi. Pengguna dapat melihat deskripsi, foto, dan lokasi kos di Google Maps. Pemilik kos juga dapat mengelola data kos mereka melalui platform ini.
